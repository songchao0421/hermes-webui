# Hermes WebUI — Stitch 重新设计提示词 (v2)

风格方向：**Cyber-noir · 赛博暗夜**
- 暗色底色 + 主题色点缀（金色/紫色/青色等用户可切换）
- 毛玻璃（glassmorphism）卡片，微光晕
- 大圆角、克制的渐变、极简图标风格
- 信息层级清晰，呼吸感充足
- 保留所有现有 DOM id 以兼容前端 JS

---

## 页面 3：聊天页 (Chat)

**核心页面。** 顺序：顶部状态栏 → 消息区 → 浮动输入栏。

### 顶部状态栏 (Top Bar)

```
  ┌─────────────────────────────────────────────────────────┐
  │  [☰]  ● 已连接          ● LOCAL  qwen2.5:7b           │
  │                                                        │
  │  (mobile: 汉堡菜单按钮  |  状态指示器  |  模型状态)     │
  └─────────────────────────────────────────────────────────┘
```

- 毛玻璃 + 底边 1px 虚线
- 左侧：移动端菜单按钮（桌面隐藏）+ 连接状态圆点 + 文字
- 右侧：模型状态 LED（绿=本地/黄=远程/蓝=路由中/红=错误/灰=未连）+ 模型名
- 右侧空间可用于未来扩展（快速操作、通知等）

## 全局设计规范

### 色彩令牌

```css
:root {
  --bg-primary: #131316;       /* 主背景 */
  --bg-surface: #1c1c1f;      /* 卡片/表面 */
  --bg-elevated: #222226;     /* 悬浮状态 */
  --bg-deep: #0e0e11;        /* 代码区/更深处 */
  --text-primary: #ffffff;
  --text-secondary: #a0a0a8;
  --text-tertiary: #6b6b73;
  --border-default: #2a2a30;
  --border-hover: #3a3a42;
  --theme-primary: #e8a849;   /* 默认金色，可切换 */
  --theme-primary-dim: #b8860b;
  --color-error: #f44;
  --color-warning: #f0a030;
  --color-success: #4caf50;
}
```

### 字体
- UI 字体：Inter 或系统无衬线（`-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`）
- 代码字体：JetBrains Mono 或 `SF Mono, Fira Code, monospace`
- 字号：18px/16px/14px/12px/11px/10px

### 圆角
- 卡片/弹窗：12-16px
- 按钮/输入框：8-12px
- 消息气泡：20px
- pill 标签：20px

### 动画
- hover：0.2s ease（shadow, border-color, transform）
- 页面切换：opacity 0.3s ease
- 消息出现：fadeIn 0.3s + translateY(8px)
- 加载中：脉冲或 skeleton

### 间距
- 页面 padding：32px 左右
- 卡片 padding：20px
- 组件 gap：16px（垂直），8px-12px（水平）
